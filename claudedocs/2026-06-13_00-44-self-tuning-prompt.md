# Цель

Сформировать инфраструктуру для интеллектуального тюнинга бота

# Дополнительные сведения

Идея в том, чтобы Claude Code мог самостоятельно модифицировать 
- код бота
- промпты
- архитектуру
чтобы улучшить его функциональность, качество и производительность.

Для этого Claude Code должен иметь возможность:
- анализировать и понимать текущую архитектуру и код бота
- генерировать новые варианты кода и промптов
- тестировать и оценивать качество и производительность новых вариантов
- выбирать лучший вариант и внедрять его в бот

А занчит у него должна быть возможность видеть что уходит в LLM, что получает и что возвращает.


# Задача

Продумай, что нужно сделать, чтобы Claude Code мог самостоятельно модифицировать код бота, 
промпты и архитектуру, чтобы улучшить его функциональность, качество и производительность.
