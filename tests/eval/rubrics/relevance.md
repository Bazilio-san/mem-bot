# relevance — релевантность ответа

Насколько каждый ответ бота отвечает именно на реплику пользователя с учётом контекста всего диалога:
по существу, без ухода в сторону, без игнорирования заданного вопроса и без ответа на вопрос, которого
не задавали.

- **5** — каждый ответ точно по сути реплики; контекст предыдущих ходов учтён; ничего важного из вопроса
  не проигнорировано.
- **4** — ответы по существу, но местами есть лишние отступления или неполный охват вопроса.
- **3** — заметные отклонения: часть вопроса проигнорирована либо ответ частично о другом.
- **2** — ответы лишь касаются темы; пользователю пришлось бы переспрашивать.
- **1** — ответы не о том, что спрашивали, либо игнорируют реплику.
