# memory_usage — использование памяти

Опирается ли бот на факты, которые обязан знать из ЭТОГО диалога (имя, цели, предпочтения, слабые темы,
созданные напоминания): упоминает их там, где они уместны, не переспрашивает уже сказанное и не отвечает
так, будто видит пользователя впервые.

- **5** — все существенные факты диалога учтены и использованы там, где они улучшают ответ;
  ничего не переспрошено.
- **4** — основные факты учтены, но какой-то второстепенный факт не использован там, где был бы уместен.
- **3** — часть существенных фактов проигнорирована либо переспрошено уже сказанное.
- **2** — бот учитывает факты лишь эпизодически; большинство ответов «без памяти».
- **1** — бот полностью игнорирует сказанное ранее.
